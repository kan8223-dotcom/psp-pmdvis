# PSP PMD Visualizer (PMDVIS)

PSPのMedia Engine (ME) を333MHzフル稼働させてPC-98のFM音源 (YM2608) をリアルタイム合成。
メインCPUはビジュアライザ描画に専念する、デュアルCPU構成のPMDプレイヤーです。

## 何ができるのか

- **PMD (.M) ファイルの再生** — pmdmini (ymfm fork) によるFM 6ch + SSG 3ch + リズム
- **Media Engineで等速FM合成** — メインCPUでは間に合わない合成処理をME 333MHzに丸投げ
- **リングバッファ256ブロック** — ME→メインCPU間の安定したPCMストリーミング
- **ピアノロール / FMPビジュアライザ** — チャンネル別のリアルタイムステータス表示
- **WAVエクスポート** — ループ回数・フェードアウト設定付きでms0:/MUSIC/に出力
- **メモステ再帰検索 (SEARCH)** — ストレージ内のどこに.Mがあっても見つけてくる
- **11パート個別ソロ/ミュート** — FM×6, SSG×3, ADPCM, Rhythm
- **○×ボタン自動判定** — 日本版/海外版を問わずシステム設定に追従
- **ループキャッシュ** — 60秒分のPCMを事前デコード、途切れないループ再生

## 動作確認済み機種

| 機種 | ファームウェア |
|------|---------------|
| PSP-1000 | 6.60 PRO-B10 |
| PSP Go (N1000) | ARK-4 CXPL |

## 必要なもの

- カスタムファームウェア導入済みのPSP
- メモリースティック (PMDファイル格納用)
- (任意) YM2608 ADPCM リズムROM — リズムチャンネルを鳴らしたい場合

## インストール

1. `EBOOT.PBP` を `ms0:/PSP/GAME/PMDVIS/` にコピー
2. (任意) YM2608 ADPCM ROMを `ms0:/PSP/GAME/PMDVIS/ym2608_adpcm_rom.bin` に配置
   このファイルがなくてもプレイヤーは動作しますが、リズムチャンネルは無音になります
3. PMD (.M) ファイルをメモステのどこかに置く
   起動後SELECTで再帰検索するか、直接 `ms0:/PSP/GAME/PMDVIS/songs/` に入れてもOK

## YM2608 リズムROMについて

YM2608 ADPCM リズムROMはヤマハ株式会社の著作物であり、本プロジェクトには**含まれていません**。
リズムチャンネルを使いたい場合は、ユーザー自身でROMを用意してください。
ROMがなくても FM + SSG は正常に動作します。

## 操作方法

### リスト画面

| ボタン | 機能 |
|--------|------|
| ○ | 曲再生 |
| × | アプリ終了 |
| ↑↓ | カーソル移動 |
| L / R | ページ送り/戻り |
| □ | WAVエクスポート設定 |
| △ | 曲削除 (確認ダイアログ) |
| START | ME ON/OFF切替 |
| SELECT | ストレージ検索 |

### 再生画面

| ボタン | 機能 |
|--------|------|
| ○ | 一時停止/再開 |
| × | リストに戻る |
| ←→ | ビジュアライザ切替 (ピアノロール⇔FMP) |
| ↑↓ | ソロカーソル移動 |
| △ | トラックミュート切替 |
| □ | ソロ全復帰 |
| L / R | 曲送り/戻り |
| L+R同時 | スクリーンショット保存 |

### WAVエクスポート設定

| ボタン | 機能 |
|--------|------|
| △ | ループ回数切替 (1⇔2) |
| □ | フェードアウト ON/OFF |
| ○ | エクスポート開始 |
| × | キャンセル |
